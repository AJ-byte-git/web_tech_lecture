Multi Column Layout

CSS allows text to be displayed in multiple columns similar to newspapers. This layout automatically flows text from one column to the next. Multi-column layout is mainly used for text-heavy content such as articles, blogs, and news pages. The number of columns can be controlled using column-count or column-width. The spacing between columns and a rule (vertical line) can also be added for better readability.
